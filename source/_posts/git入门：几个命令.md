Git入门
作为新手，说说几个关于git的命令
1.git init
  该命令是用来初始化一个git仓库；
  执行完会在git仓库生成一个.git文件
  ![git_init.png](https://i.loli.net/2018/05/24/5b06530eaa084.png)

2.git add
  该命令用于将文件添加到缓存
  添加前是以??开头，添加后是以A开头
  ![git_add.png](https://i.loli.net/2018/05/24/5b06530e989d4.png)

3.git commit -v
  该命令用于将缓存区的文件添加到仓库中
  ![git_commit.png](https://i.loli.net/2018/05/24/5b06530ea9385.png)
  

完了，
继续学吧~~~
